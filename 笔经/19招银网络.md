* 下面这一段代码并不能输出1,2,3,4,5，解释下原因，并且修正下函数，使之输出正确的结果

```
  for (var i=1;i<=5;i++) {
    setTimeout(function timer(){
      console.log(i);
    },i*1000);
  }
  
  //以每秒一次的频率输出5个6
```

这题考的闭包，在《你不知道的JavaScript》上看到一个很好的解决方法，利用块作用域。看下面
```
for (let i=1;i<=5;i++) {
    setTimeout(function timer(){
      console.log(i);
    },i*1000);
  }
```
